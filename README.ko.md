# sch-customize

내러티브 중심의 TRPG 세션을 위해 Foundry VTT의 채팅 경험을 다듬는 모듈입니다. 잡담 스타일링, 채팅 로그 아카이브, 발화자 고정, 채팅 외관 세밀 조정 기능을 제공합니다.

> Foundry VTT v12 – v13 호환 (v14 지원 작업 중).

<!-- 📸 [대표 스크린샷] — 채팅 사이드바 전체를 한 장에 담은 인상적인 샷:
     - 여러 잡담이 그룹화된 모습 (서로 다른 유저, 다른 색상)
     - 입력창 위에 발화자 바
     - 일반 채팅 메시지도 섞여 있는 자연스러운 상태
     권장 폭: ~400px. 모듈을 처음 보는 사람에게 가장 강한 인상을 주는 자리. -->

## 주요 기능

### 💬 잡담 (Chitchat / Side Talk)

캐릭터 대사·롤과 시각적으로 구분되는 사담을 입력할 수 있습니다. 같은 사용자의 연속된 잡담은 자동으로 한 덩어리로 묶입니다(설정에서 끌 수 있음).

<!-- 📸 [잡담 그룹화] — 같은 유저의 연속 잡담 3-4개를 클로즈업한 샷.
     top/middle/end 둥근 모서리 그룹 효과가 보이도록.
     선택 사항: 색상이 다른 두 유저의 잡담을 나란히 배치하여
     유저 색상 배경 테마를 함께 보여주기. -->

- `/pt`, `!`, `` ` ``, 또는 직접 지정한 커스텀 문자(기본 `/p`)로 입력 시작
- 연속 잡담은 둥근 모서리로 그룹화 (top / middle / end) — 토글 가능
- 사용자 색상을 배경으로 사용, 밝기 조절 가능
- 메시지 분류를 **OOC** 또는 **기타(Other)** 중 선택
- 잡담 내부에서 마크다운 취소선(`~문자~`) 지원
- 화자 이름 표기 후 줄바꿈 옵션
- 폰트 크기, 진하기, 왼쪽 여백 조절

### 🧵 일반 채팅 메시지 그룹화

잡담과 동일한 둥근 `top / middle / end` 묶음 처리가 일반 채팅 메시지에도 적용됩니다. 같은 플레이어의 연속 메시지이면서 메시지 스타일이 동일할 때 자동으로 합쳐집니다. 기본 활성화되어 있으며, 설정에서 끌 수 있습니다.

### 🦊 발화자 바 (Speaker Bar)

채팅 입력창 위에 현재 어떤 화자로 발화될지를 표시합니다.

<!-- 📸 [발화자 바 — 두 상태] — 좌우 비교 또는 GIF 한 장:
     (A) 잠금 해제 상태: 열린 자물쇠, 선택된 토큰의 초상화 + 이름
     (B) 잠금 상태: 닫힌 금색 자물쇠, 같은 초상화
     클릭하여 잠그는 순간을 보여주는 GIF가 가장 효과적. -->

- 선택된 토큰을 자동으로 따라감
- 토큰이 없으면 할당된 캐릭터 → 사용자 이름 순으로 폴백
- 초상화 클릭 시 해당 액터 시트 열기
- 자물쇠 아이콘으로 **발화자 고정** — 잠근 후에는 토큰 선택을 바꿔도 화자가 유지됨
- 잠금 상태는 새로고침/재접속 후에도 유지 (사용자 flag로 저장)
- 발화자 바 자체를 설정에서 끌 수 있음 (기본 채팅 UI를 유지하고 싶을 때)
- **발화자 즐겨찾기**: 자주 쓰는 발화자를 바 아래 칩으로 고정합니다. 칩을 클릭하면 그 화자로 고정(lock)이 즉시 전환되고, 칩에 마우스를 올려 ×로 삭제하며, **[+]**로 현재 발화자를 추가합니다. 여러 NPC를 오가는 GM에게 유용합니다. 칩 줄은 설정에서 켜고 끌 수 있습니다.

### 📥 채팅 로그 아카이브

<!-- 📸 [아카이브 다이얼로그] (선택) — "채팅 로그 다운로드" 설정 메뉴 또는
     내보낸 HTML 결과물 미리보기. 위 시각적 기능들보다는 우선순위 낮음. -->

- **채팅 로그 다운로드**: 현재까지의 채팅 로그를 이미지 동봉 HTML 파일로 내보내기
- **채팅 로그 표시**: 전체 채팅 로그를 새 창에서 열기
- 귓속말 포함/제외 옵션
- 귓속말 숨김 옵션: 회색 자리표시자로 가리고 클릭 시 내용 표시

### 🎨 외관 조정

<!-- 📸 [설정 패널] (선택) — 외관 슬라이더들이 보이는 모듈 설정 화면 캡처.
     설정 가능 항목을 미리 확인하고 싶은 사용자에게 유용. -->

- 기본 채팅 폰트 크기 (14 – 30 px)
- 잡담 폰트 크기 (10 – 30 px)
- 잡담 폰트 색상 진하기 (0 – 1)
- 잡담 왼쪽 여백 (0 – 40 px)
- 잡담 배경 밝기 (0 – 1)

## 설치

### 권장 — Manifest URL 사용

Foundry VTT에서 **모듈 관리** → **모듈 설치** → 아래 manifest URL 붙여넣기:

```
https://github.com/JBGeum/sch-customize/releases/latest/download/module.json
```

### 수동 설치

[Releases 페이지](https://github.com/JBGeum/sch-customize/releases)에서 최신 `sch-customize.zip`을 다운로드한 뒤 `Data/modules/sch-customize/`에 압축을 풀고 Foundry를 재시작합니다.

### 의존 모듈

**[Chat Commands](https://gitlab.com/woodentavern/foundryvtt-chat-command-lib)** 모듈이 필요합니다. 최근 버전 Foundry에서는 의존성 자동 설치가 작동합니다.

## 사용법

### 잡담 입력

메시지 맨 앞에 다음 중 하나를 입력합니다:

```
/pt 이건 사담이에요.
/p  커스텀 문자로 시작한 사담.
!   한 글자로 빠르게.
`   백틱도 가능.
```

한 글자짜리 커스텀 trigger는 모듈 설정에서 자유롭게 변경할 수 있습니다.

### 발화자 고정

<!-- 📸 [고정 흐름] (선택 — GIF가 가장 효과적) — 짧은 애니메이션:
     1. 캔버스에서 토큰 선택 → 발화자 바 갱신
     2. 자물쇠 클릭 → 금색으로 변함
     3. 다른 토큰 선택 → 발화자 바는 원래 토큰 유지
     GIF가 부담스러우면 3컷 정지 이미지도 OK. -->

1. 캔버스에서 토큰 선택 (선택된 토큰이 없으면 안내 메시지가 표시됩니다)
2. 발화자 바의 **🔓 열린 자물쇠** 클릭
3. 자물쇠가 금색으로 변하며 발화자 고정됨
4. 이후 입력하는 메시지는 모두 고정된 토큰의 화자로 나감 (다른 토큰을 선택해도 유지)
5. **🔒 닫힌 자물쇠**를 다시 클릭하면 해제

### 채팅 로그 보관

**설정 → 모듈 설정 → sch-customize**에서:
- **채팅 로그 다운로드**: HTML 파일로 내보내기
- **채팅 로그 표시**: 새 창에서 열기

## 호환성

- **Foundry VTT**: v12, v13 검증. v14 지원 작업 중 — 코드 내부는 이미 v13+의 `renderChatMessageHTML` / `ChatMessage#renderHTML` / `CHAT_MESSAGE_STYLES` API를 사용하며, v12를 위한 폴백 분기를 함께 둡니다.
- **시스템**: 시스템 비의존. 내러티브 중심 사용 목적으로 설계되었으며, 주로 DnD5e와 generic 시스템에서 테스트됨
- **다른 모듈**: 대부분의 채팅 확장 모듈과 호환. 채팅 배경색을 변경하는 모듈을 함께 사용할 경우 잡담 밝기 설정값을 조절할 필요가 있을 수 있습니다.
